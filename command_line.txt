# Lệnh makemigrations để tạo các file migration, File migration này nhằm xử lý sẽ tác động đến database

# Lệnh migrate để áp dụng các thay đổi đó vào database.
# python manage.py migrate

# B1: thay đổi trong folder static của project admin
# B2: python manage.py collectstatic

# Lệnh tạo superuser
# python manage.py createsuperuser

# python manage.py startapp <app-name>
